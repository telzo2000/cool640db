# cool640db



![](img/img00001.jpg)

## What is cool640db?

cool640db is small keyboard with 2 trackballs.
<br>
cool640db is splite keyboards.
<br>
cool640db use seeed xiao ble,run zmk_firmware.
<br>
<br>
